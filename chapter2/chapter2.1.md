2.1-3

```c
SEARCH(A,v)
   for i = 1 to A.length
      if A[i] == v
         return i
   return NIL
```

循环不变式
> 在每次循环开始时， 子数组A[1,..,i-1]的值不包含v

初始化 
> 开始时，数组为空，明显为真 

保持
> 如果A[i] == v,我们返回i，这是正确的。否则A[i]!=v,我们已知A[1,...,i-1]都不等于v,所以下一次循环开始时仍然满足循环不变式

终止
> 当i > A.length ,循环终止时，A[1,...,i]都不等于v，所以返回NIL是正确的
